""" Projet de blindtest pour une soirée étudiante en ligne """

Code généré par Claude IA

""Fonctionnalitées : ""
* importer un fichier csv
* reconnaitre le titre, le chanteur, l'année et les catégories
* pouvoir choisir une ou plusieurs catégorie(s) pour une session de jeu
* choisir la durée de l'extrait
* choisir le temps de pause entre les morceaux
* choisir le nombre de morceau
* pouvoir les passer en mode aléatoire

Ce code évoluera en fonction des bugs rencontrés :
* actuellement, les fonctionnes `tout sélectionner` et `tout désélectionner`sélectionne chaque catégorie
* la catégorie `histoire d'mour`reste bloquée
