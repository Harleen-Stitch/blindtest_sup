Comment obtenir un Client ID ?
1

Créer une app Spotify

Va sur developer.spotify.com/dashboard → Log in → Create App

2

Configurer l'app

Nom : "Blindtest" (n'importe quoi). Redirect URI : l'URL exacte de cette page HTML (ex: file:///chemin/vers/blindtest_v2.html ou ton URL en ligne). Cocher "Web API".

3

Copier le Client ID

Dans les Settings de ton app → copie le Client ID → colle-le ici. Pas besoin du Client Secret.

4

Compte premium requis

Le Web Playback SDK (lecture complète) nécessite Spotify Premium. Sans Premium, l'app utilise les previews 30 secondes via l'API publique.
